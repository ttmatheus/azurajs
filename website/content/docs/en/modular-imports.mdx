# Modular Imports üì¶

AzuraJS supports modular imports for better tree-shaking and smaller bundle sizes. Import only what you need!

<Callout type="info">
**JavaScript Note:** All imports work the same in JavaScript! Just remove type imports and TypeScript-specific syntax.
</Callout>

## Main Package

Import everything from the main package:

### TypeScript

```typescript
import { AzuraClient } from "azurajs";
```

### JavaScript

```javascript
const { AzuraClient } = require("azurajs");
```

## Decorators

Import decorators separately:

### TypeScript

```typescript
import { Controller, Get, Post, Put, Delete, Patch } from "azurajs/decorators";
import { Body, Param, Query, Headers, Req, Res } from "azurajs/decorators";
import { applyDecorators } from "azurajs/decorators";
```

### JavaScript

```javascript
// Not applicable - Decorators are a TypeScript feature
// Use the functional API instead:
const { AzuraClient } = require("azurajs");
const app = new AzuraClient();
app.get("/", ({ req, res }) => { /* ... */ });
```

## Middleware

Import middleware utilities:

### TypeScript

```typescript
import { createLoggingMiddleware } from "azurajs/middleware";
```

### JavaScript

```javascript
const { createLoggingMiddleware } = require("azurajs/middleware");
```

## Types

Import TypeScript types:

```typescript
import type { RequestServer, ResponseServer } from "azurajs/types";
// Or use modular import
import type { ConfigTypes } from "azurajs/config";
```

## Plugins

Import plugins separately:

```typescript
// CORS Plugin
import { cors } from "azurajs/cors";

// Rate Limiting
import { rateLimit } from "azurajs/rate-limit";
```

## Utilities

Import utilities as needed:

### TypeScript

```typescript
// Logger
import { logger } from "azurajs/logger";

// HTTP Error
import { HttpError } from "azurajs/http-error";

// Validators
import { validateDto, validateSchema } from "azurajs/validators";

// Cookies
import { parseCookiesHeader, serializeCookie } from "azurajs/cookies";
```

### JavaScript

```javascript
// Logger
const { logger } = require("azurajs/logger");

// HTTP Error
const { HttpError } = require("azurajs/http-error");

// Validators
const { validateDto, validateSchema } = require("azurajs/validators");

// Cookies
const { parseCookiesHeader, serializeCookie } = require("azurajs/cookies");
```

## Router

Import router separately:

```typescript
import { Router } from "azurajs/router";
```

## Complete Example

### TypeScript

```typescript
// Main imports
import { AzuraClient } from "azurajs";
import { applyDecorators } from "azurajs/decorators";

// Decorators
import { Controller, Get, Post, Body, Param, Res } from "azurajs/decorators";

// Middleware
import { createLoggingMiddleware } from "azurajs/middleware";

// Utilities
import { HttpError } from "azurajs/http-error";

// Types
import type { ResponseServer } from "azurajs/types";
import type { ConfigTypes } from "azurajs/config";

@Controller("/users")
class UserController {
  @Get("/")
  getAll(@Res() res: ResponseServer) {
    res.json({ users: [] });
  }

  @Get("/:id")
  getOne(@Param("id") id: string, @Res() res: ResponseServer) {
    if (id === "0") {
      throw new HttpError(404, "User not found");
    }
    res.json({ id, name: "User" });
  }

  @Post("/")
  create(@Body() body: any, @Res() res: ResponseServer) {
    res.status(201).json({ id: Date.now(), ...body });
  }
}

const app = new AzuraClient();
app.use(createLoggingMiddleware(app.getConfig()));
applyDecorators(app, [UserController]);

await app.listen(3000);
```

### JavaScript

```javascript
// Main imports
const { AzuraClient } = require("azurajs");

// Middleware
const { createLoggingMiddleware } = require("azurajs/middleware");

// Utilities
const { HttpError } = require("azurajs/http-error");

const app = new AzuraClient();
app.use(createLoggingMiddleware(app.getConfig()));

// Routes
app.get("/users", ({ req, res }) => {
  res.json({ users: [] });
});

app.get("/users/:id", ({ req, res }) => {
  const { id } = req.params;
  if (id === "0") {
    throw new HttpError(404, "User not found");
  }
  res.json({ id, name: "User" });
});

app.post("/users", ({ req, res }) => {
  res.status(201).json({ id: Date.now(), ...req.body });
});

app.listen(3000);
```

## Import Comparison

### Before (‚ùå No longer works)

```typescript
// This example does NOT work anymore in version 2.4.0+
import {
  AzuraClient,
  Controller,
  Get,
  Post,
  Body,
  HttpError,
  logger,
  cors,
  rateLimit
} from "azurajs";
```

**Bundle size:** ~50KB (everything included)

### After (‚úÖ Correct - Modular imports)

```typescript
import { AzuraClient } from "azurajs";
import { Controller, Get, Post, Body } from "azurajs/decorators";
import { HttpError } from "azurajs/http-error";
import { logger } from "azurajs/logger";
import { cors } from "azurajs/cors";
import { rateLimit } from "azurajs/rate-limit";
```

**Bundle size:** ~15KB (only what you need)

## Migration Guide

If you're using older imports, here's how to migrate:

```typescript
// ‚ùå Old way (no longer works)
import { validateDto } from "azurajs";

// ‚úÖ New way (correct)
import { validateDto } from "azurajs/validators";

// ‚ùå Old way (no longer works)
import { HttpError } from "azurajs";

// ‚úÖ New way (correct)
import { HttpError } from "azurajs/http-error";

// ‚ùå Old way (no longer works)
import { cors } from "azurajs";

// ‚úÖ New way (correct)
import { cors } from "azurajs/cors";
```

<Callout type="warn">
  **Important!** Starting from version 2.4.0, you MUST use modular imports. The main package (`azurajs`) only exports `AzuraClient` and basic types. Specific features like decorators (including `applyDecorators`), logger, validators, http-error, cors, etc. must be imported from their respective modules.
</Callout>

| Rate Limit | `azurajs/rate-limit` | Rate limiting |
| Cookies | `azurajs/cookies` | Cookie utilities |
| Validators | `azurajs/validators` | Validation functions |
| Logger | `azurajs/logger` | Logging utility |
| HTTP Error | `azurajs/http-error` | Error class |
| Types | `azurajs/types` | Common types |
